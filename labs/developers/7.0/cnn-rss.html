CNN RSS

During this lab you will create a simple RSS reader using the Ionic Framework, MobileFirst Platform Foundation and Atom code editor as the main IDE with MFP Command Line Interface; get used to work with MobileFirst Platform Foundation adapters, MobileFirst Platform Foundation Analytics, MobileFirst Platform Foundation Operational Console and more.

Prepare hybrid project using Ionic

Description: Using the Ionic framework, create a simple project, migrate it to MobileFirst Platform and deploy it to the MobileFirst Server.
Time to complete: 30 minutes

Build, deploy, pack and push the app to Android Emulator

Description: Create an Android environment, build and deploy the app to MobileFirst Server, prepare the Android .apk and install it on the Android Emulator.
Time to complete: 15 minutes

Establish server connectivity and use Direct Update

Description: Using client-side API, implement server connection from the app and by using Direct Update deliver refreshed web resources to the application.
Time to complete: 25 minutes

Implement application logic

Description: Using Ionic and AngularJS prepare the news application structure, and using Direct Update push updates to the application.
Time to complete: 25 minutes

Implement an HTTP adapter to retrieve data from an RSS feed

Description: Develop the integration part of the news application, by using HTTP adapter and RSS feed. Grab articles and display them in the mobile application.
Time to complete: 40 minutes

Add news categories, search capabilities and pull-to-refresh option

Description: Add news categories buttons in sidemenu to grab regional news, add search using filtering in angular and pull-to-resfresh functionality. Time to complete: 25 minutes

Modify application UI

Description: Modify XSL file to grab image thumbnails and full article images. Add CSS styles to improve the applicationâ€™s look & feel.
Time to complete: 30 minutes

Finalize application

Description: Replace default icons and splash screen for Android application. Fix application components loading order and install final .apk in the mobile device.
Time to complete: 25 minutes
