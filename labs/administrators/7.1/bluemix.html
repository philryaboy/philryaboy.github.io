Overview

Follow the instructions below to learn how to install MobileFirst Platform Server on IBM Cloud.

For this purpose, the MobileFirst Platform Server will be setup in a Docker container image, and pushed to an IBM Container on IBM Cloud, including MobileFirst Platfrom Operational Analytics. An Ubuntu 14.04 image will be used to perform the installation steps.

Pre-requisites:

Follow the 7.1 Labs required setup to install Ubuntu, install the MobileFirst CLI and register at IBM Cloud

Setup prerequisites for MobileFirst Platform installation on IBM Cloud

Description: Download the MobileFirst Platform evaluation image, setup Docker and VirtualBox, install Cloud Foundry command-line interface and IBM Containers extension.
Time to complete: 25 minutes
Lab files:
  • {% highlight bash %} wget https://bootstrap.pypa.io/get-pip.py sudo python get-pip.py sudo pip install pyopenssl ndg-httpsclient pyasn1 sudo pip install requests[security] {% endhighlight %}
  • VirtualBox installation using command-line on Ubuntu

Install MobileFirst Platform Analytics container on IBM Cloud

Description: Setup and configure MobileFirst Platform analytics image in docker container and push it to IBM Cloud.
Docker machine environment setup: eval "$(docker-machine env default)"
Time to complete: 45 minutes

Install MobileFirst Platform Server 7.1 on IBM Cloud

Description: Setup and configure MobileFirst Platform Server image in docker container with test runtime environment and push it to IBM Cloud.
Docker machine environment setup: eval "$(docker-machine env default)"
Time to complete: 45 minutes

Verify MobileFirst Platform 7.1 installation on IBM Cloud

Description: Deploy test application and adapter and verify platform setup using it.
Time to complete: 10 minutes
