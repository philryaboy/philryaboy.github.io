## 概説
{: #overview }
{{ site.data.keys.product_full }} のシステム要件には、オペレーティング・システム、SDK、およびその他のソフトウェアが含まれます。

{{ site.data.keys.product }} には、製品を正常にインストールおよび構成するために満たす必要のあるいくつかのシステム要件があります。 システム要件には、以下の項目が含まれます。

* {{ site.data.keys.product }} をサポートするオペレーティング・システム (モバイル・デバイス・オペレーティング・システムを含む)
* サポートされる  Software Development Kit (SDK)
* {{ site.data.keys.product }} で必要であるかまたはサポートされるアプリケーション・サーバー、データベース管理システム、およびその他のソフトウェア
