## Übersicht
{: #overview }
Zu den Systemvoraussetzungen für die {{ site.data.keys.product_full }} gehören
Betriebssysteme, SDKs und andere Software. 

Für die {{ site.data.keys.product }} gelten
einige Systemvoraussetzungen, die erfüllt sein müssen, um das Produkt erfolgreich installieren und konfigurieren zu können. Unter anderem gelten
folgende Systemvoraussetzungen: 

* Betriebssysteme mit Unterstützung der {{ site.data.keys.product }}, einschließlich
Betriebssystemen für mobile Geräte
* Unterstützte Software Development Kits (SDKs)
* Anwendungsserver, Datenbankmanagementsysteme und andere Software, die für die
{{ site.data.keys.product }} erforderlich ist oder vom Produkt unterstützt wird
