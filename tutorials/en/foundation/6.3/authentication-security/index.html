Overview

The Authentication and security category is meant for all developers, whether aiming for native or hybrid applications.

In this category, the developer will learn:

  • The concepts of authentication and security in MobileFirst applications,
  • Implement various application authentication methods such as: form-base, adapter-based, custom-based,
  • As well as configure the application to use device provisioning and application authenticity protection methods, and more.
