Logging example for native iOS (Objective-C)

Outputs to Xcode console

{% highlight objc %} #import "OCLogger.h" + (int) sum:(int) a with:(int) b{ int sum = a + b; OCLogger* mathLogger = [OCLogger getInstanceWithPackage:@"MathUtils"]; NSString* logMessage = [NSString stringWithFormat:@"sum called with args %d and %d. Returning %d", a, b, sum]; [mathLogger debug:logMessage]; return sum; } {% endhighlight %}

API calls for specific tasks

Log capture is enabled by default. To turn log capture on or off:

OCLogger.setCapture(NO)

The default capture level is DEBUG in development and FATAL in production. To control the capture level (verbosity):

OCLogger.setLevel(OCLogger_DEBUG)

Log sending is enabled by default. To turn automatic log sending on or off:

OCLogger.setAutoSendLogs(NO)

For more information about the OCLogger API, review the "Objective-C Client-Side API for Native iOS Apps" user documentation topic
