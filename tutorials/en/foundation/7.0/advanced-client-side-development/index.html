Overview

The Advanced client-side development category is meant for all developers, whether aiming for native or hybrid applications.

  • Hybrid application developers can learn how to implement different MobileFirst features, such as: UI controls, skins, offline access, translation and encryption of sensitive data
  • Native and Hybrid application developers also learn: using the MobileFirst CLI to work outside of MobileFirst Studio, as well as how to handle log collection from the client application
