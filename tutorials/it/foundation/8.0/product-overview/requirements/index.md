## Overview
{: #overview }
System requirements for {{ site.data.keys.product_full }} include operating systems, SDKs, and other software.

{{ site.data.keys.product }} has a number of system requirements that must be met for you to install and configure the product successfully. The system requirements include the following items:

* Operating systems that support {{ site.data.keys.product }}, including mobile device operating systems
* Supported software development kits (SDKs)
* Application servers, database management systems, and other software that are required or supported by {{ site.data.keys.product }}
