Class LogInstance 

LogInstance
Description:

Logger with its own context (also called status or state). New instances are created with WL.Logger.create.

Example:
var logger = WL.Logger.create({pkg: 'myapp'});
logger.debug('Hello world');
//[myapp] Hello world
