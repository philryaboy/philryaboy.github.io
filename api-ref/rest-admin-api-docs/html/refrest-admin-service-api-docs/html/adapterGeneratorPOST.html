Adapter Generator (POST)

Description

Generates an adapter from an OpenAPI (a.k.a. Swagger) specification of a microservice.

Roles

Users in the following roles are authorized to perform this operation:

  • mfpadmin
  • mfpdeployer

Method

POST

Path

mfpadaptergenerator/adapterapi/generate

Form parameters

pBody
OpenAPI specification file (.json/.yaml).
includesrc
Optional. If source code is required, this parameter should be set to true.

Consumes

multipart/form-data

Produces

application/octet-stream

Example

http://www.example.com/mfpadaptergenerator/adapterapi/generate

Responses

Downloads an Adapter File/Adapter Source Code

Errors

403 
Unauthorized Access
412 
Precondition Failed
500 
An internal error occurred
